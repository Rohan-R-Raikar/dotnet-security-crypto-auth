# dotnet-security-crypto-auth

This repository contains multiple C#/.NET projects focused on **security, authentication, and cryptography**.

## Projects

1. [JWTcontinueIfSessionActive](JWTcontinueIfSessionActive) - Demo for JWT session handling
2. [JwtAuthDotNet9](JwtAuthDotNet9) - JWT authentication with .NET 9
3. [HmacAuthServer](HmacAuthServer) - HMAC-based authentication server
4. [HmacAuthClient](HmacAuthClient) - HMAC-based client
5. [TwoFactorAuthDemo](TwoFactorAuthDemo) - Two-factor authentication demo
6. [OpenIDConnectOIDC](OpenIDConnectOIDC) - OpenID Connect implementation
7. [CertAuthTestSln](CertAuthTestSln) - Certificate-based authentication
8. [AesEncryptionDemo](AesEncryptionDemo) - AES encryption example
9. [ConsoleAppForDUKPT](ConsoleAppForDUKPT) - DUKPT encryption demo
10. [PEKtodecryptedTrack2Data](PEKtodecryptedTrack2Data) - Decrypt Track2 data using PEK
11. [TR31 Version D](TR31 Version D) - TR31 cryptography standard implementation
